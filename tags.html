What are HTML tags?

HTML tags are used to define the structure and content of a web page. Tags are enclosed in angle brackets, like <tag>. Most tags come in pairs, such as <p> and </p>.

Most typical HTML Tags

Below are some commonly used tags:

Tag Purpose Example
<h1> to <h6> Defines headings. <h1>This is a heading</h1>
<p> Defines a paragraph. <p>This is a paragraph.</p>
<a> Creates hyperlinks. <a href="url">Link</a>

Inline Elements

Inline elements do not break the flow of text. Examples include:

  • <em>: Adds emphasis (ex., emphasis).
  • <strong>: Indicates importance (ex., important).
  • <code>: Displays code snippets (ex., <html>).
